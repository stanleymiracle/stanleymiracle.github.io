TeX 链接

  1. CTEX 和 CTEX 的讨论区

    中国 TeX 用户的主力站点。可以在这里讨论所有有关 TeX 的话题。这里有一个 CTAN 的镜像，可以下载很多程序。

  2. TUG(TeX Users Group)

    TUG 就是全世界 TeX 用户的组织。负责发行 TUGboat 杂志，组织会议，发行 TeXLive CD 等工作。他们还负责管理 CTAN.

  3. CTAN (the Comprehensive TeX Archive Network)

    这里就是所有 TeX 宏包，发行，文档的聚集地！可以搜索你所需要的，包括各种发行，各种宏包，文档……
