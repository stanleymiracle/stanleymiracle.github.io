Java plugin 的安装注意事项。

很多人安了 Java plugin 之后都会 crash，原因是它们把那个 libjavaplugin_oji.so 直接拷贝到了 Mozilla 的 plugin 目录。 应该做一个符号连接才行！

Mozilla 如何嵌入的显示 PDF?

把 Acrobat Reader 的 nppdf.so 拷贝到 Mozilla 的 plugins 目 录就行了。nppdf.so 一般在 Acrobat Reader 目录的 Browsers/intellinux/ 目录下。

装了 nppdf， 但是还是显示不了 PDF 文件。

这个原因很有可能是你的 ~/.Xdefaults 文件里有这么一行： 

*.preeditType: OverTheSpot

这个 * 匹配了所有的程序，所以 Acrobat reader 的 plugin 不接 受这个输入模式的话就会出错。

如何嵌入显示 PS, dvi 文档？

plugger，四海一家的解决方案。看看它嵌入显示的 gv 和 dvi。

Mozilla 安装的 extension 放到哪里去了？

一般在 ~/.mozilla/.../chrome 下。有一个文件叫 installed-chrome.txt 记录了安装的东西的信息。比如以下几行： 

content,install,url,jar:resource:/chrome/spade.jar!/content/spade/
content,install,url,jar:resource:/chrome/adblock.jar!/content/
content,install,url,jar:resource:/chrome/aggreg8.jar!/content/

就是告诉 Mozilla 安装了 spade, 在 spade.jar 文件里。
