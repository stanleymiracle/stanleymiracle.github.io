第三章

安装

3.1  系统要求

3.1.1  Scheme 解释器

你首先要有一个 Scheme 解释器，你可以用 MzScheme, guile, bigloo, chez, chicken, gambit, kawa, GNU/MIT scheme, petite, plt, pscheme, scheme48, scm, scsh, stk, sxm, umbscheme, ... 都行。

我个人推荐 MzScheme. 因为它遵守 R5RS，功能完善。而且因为它是 tex2page 的作者使用的，不会出现不兼容问题。

或者你可以不用 Scheme 而用 Common Lisp 解释器。不过估计除非 你是一个老牌 Common Lisp 用户，否则你还是用 Scheme 比较好。

3.1.2  TEX系统 (插入数学公式必须)

如果你要在文档里使用数学公式，插入 eps 图形，那么你需要一套 TEX系统。因为 tex2page 需要由 TEX来处理数学公式。如果你 不需要插入 TEX公式，那么就不需要 TEX系统。

你需要注意的是要把 tex 程序的路径加入系统的 PATH 环境变量， 否则 tex2page 可能找不到你的 TEX系统。

3.2  安装

tex2page 的安装很简单。 解压之后，把 tex2page 脚本拷贝到 PATH 中的一个目录，比如 /usr/local/bin.

如果你的 Scheme 解释器是 MzScheme，安装就完成了。如果不是， 或者你用 Common Lisp, 那就自己看看 INSTALL 文件吧。

如果你使用了 tex2page 提供的特殊命令比如 ，而需要 TEX和 能原封不动处理你的文档。那么你需要把所有的 tex2page.tex 和 tex2page.sty 两个文件拷贝到你的 TEX目录树中。比如我把它们拷贝到了

/usr/local/texlive/texmf-local/tex/latex/tex2page/
