列表结构

8.1  enumerate

你可以用 enumerate 环境来得到有计数的列表。 下面这个列表用一个数字列表列举了一些基本的 输入字符时的基 本要素。

  1. Hi, 今天是 五月 11, 2005。你可以用 \today 输入今天的日 期。

  2. 左边的双引号要用两个反引号 ` 表示，右边的双引号 用两个单引号表示 '.

  3. 这些字符 $ & % # _ { } ~ ^ 要这样输入：

    \$ \& \% \# \_ \{ \} \~{} \^{} 

  4. 你可以用 \verb|something| 原封不动的输入某些特 殊的 TEX符号。比如 \verb|\documentclass{article}| 得到 \documentclass{article}.

  5. Use \textbf{something} to enter something with font Bold Weight.

  6. Use \textsf{something} to enter something with font Sans Serif.

  7. \bfseries can change all the text follows to bold face, until the group ends.

  8. 你可以用改变字体大小的命令 {\tiny tiny \scriptsize scriptsize \footnotesize footnotesize \small small \normalsize normalsize(default) \large large \Large Large \LARGE LARGE \huge huge \Huge Huge.}

    结果是：

    tiny scriptsize footnotesize small normalsize(default) large Large LARGE huge Huge.

  9. TEX里有三种不同的横线: emdash, endash 和 hyphen。

    • emdash 就是“破折号”，用3根-(减号)表示: He lay there -- dead.

    • endash 用来连接一个范围，用2根减号表示: from 10-12

    • hyphen 就是连字号，就是一根减号: state-of-the-art.

  10. 不要使用 ... (三个点) 来表示省略号。因为它们挤得很紧，不 好看。在正文中你应该用 \ldots 表示省略号。看看这个 ...是不是好看多了？

    其实 TEX里有好几种不同的省略号，你应该在适当的地方使用适当 的省略号，这样你的文档才会好看。

    1. \ldots (...): 用于位置较低的符号，比如 , , ..., .

    2. $\cdots$ (): 用于位置较高的符号，比如 .

    3. $\vdots$ () 和 $\ddots$ (), 主要用于矩阵里的省略表示，比如：

    4. \dotfill 用于填充一行，就像某些目录里 的那些点线。

8.2  列表的样式

enumerate 的各种样式可以用 CSS 设定，比如下面这些 CSS 样式 定义了：第1级数字列表用阿拉伯数字表示，第2级用小写字母，第3 级用罗马数字，第4级用大写字母。

ol { 
     list-style-type: decimal; 
     } 
 
ol ol { 
        list-style-type: lower-alpha; 
        } 
 
ol ol ol { 
           list-style-type: lower-roman; 
           } 
 
ol ol ol ol { 
              list-style-type: upper-alpha; 
              } 

回头看看我的第二层数字列表，是不是显示为小写字母编号 a., b., c., ...? 你可以试试加入第 3 层列表是不是显示为 i., ii., iii, iv, ...
